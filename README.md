# Nesine.com Case Çalışması

### Projeyi başlatmak için terminal üzerinde

```html
git clone https://github.com/uyilboga/nesine-case.git
cd nesine-case
```

npm için
```html
npm install
npm run start
```

yarn için
```html
yarn install
yarn start
```

### Özellikler
- Webpack
- Babel
- React 18
- React Router Dom 6.x
- Ecmascript6
- Eslint (Airbnb), Prettier, Husky
- Sass
- Context API
- React Hooks
